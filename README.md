# Next JS Auth

Simple next auth with JWT and cookies. No DB or RPC.

```bash
yarn dev
```

## Code History

The code in this repository is based on the
[Next.js JWT authentication httpOnly cookie](https://youtu.be/T6fRWZWrJzI)
video.

```bash
npm create next-app
cd next-auth
yarn add jsonwebtoken cookien
```
